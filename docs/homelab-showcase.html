TrueNAS Scale
ZFS storage platform. RAIDZ1 pool backing PBS datastore, shared VM storage, and app persistent volumes via NFS.

Pi-hole
Whole-home DNS filtering. Primary on util01, secondary on qdev01. Automatic failover via router DHCP.
Planned · M6
Proxmox Backup Server
VM snapshot management. Datastore on nas01 via NFS. Cold tier to Backblaze B2.

Prometheus + Grafana
Metrics collection and dashboards across all hosts and AWS EC2. Hosted on mon01 (pve02) — Proxmox HA live migration ensures zero-downtime failover to pve01 on node failure.
Planned · M7
Loki + Promtail
Log aggregation from all managed hosts. Container stdout and journald forwarding.
Planned · M7
Alertmanager
Alert routing and notification. SLO burn rate rules across all services.
Planned · M7
